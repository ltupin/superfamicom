## 🇫🇷 Français | 🇬🇧 English | 🇯🇵 日本語

### 🇫🇷 Présentation

Ce dépôt contient une copie du fichier [`listing.csv`](docs/data/listing.csv) ainsi qu’un site statique dans le dossier `docs/`.

Le fichier CSV a été constitué à partir de plusieurs sites internet. Il a fait l’objet de nombreuses corrections, et les informations qu’il contient ont été vérifiées dans la grande majorité des cas.

Cependant, des erreurs peuvent encore être présentes. Les contributions sont donc les bienvenues : vous pouvez proposer une modification via une pull request sur ce dépôt.

Si vous ne savez pas comment procéder mais souhaitez tout de même contribuer, vous pouvez me contacter à [laurent.tupin@gmail.com](mailto:laurent.tupin@gmail.com).
